# 云瞳

自用暴力扫描器，`python -m yuntong --url 'http://xxxx.com'`

主要特性
- 提取HTML中的注释、表单、链接和HTTP异常Header
- 爆破泄漏文件（使用缩减的dirsearch字典）和请求参数（使用arjun字典）
- 对每一个请求参数尝试进行sql, php/shell命令执行, php include和ssti fuzz
- 对401页面进行密码爆破
